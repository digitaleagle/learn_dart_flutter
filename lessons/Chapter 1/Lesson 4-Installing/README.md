# Concepts to Learn

## Android Studio

Android Studio is my favorite IDE for Flutter. An IDE is basically a text editor on steroids. It offers
many tools that make write code easier such as coloring the code and underlining syntax errors in the code.

Android Studio and Visual Studio Code are both recommended IDEs. You can use either one. I would recommend
installing both, explore both, and make your own decision.

(Follow the Windows instructions unless you have another operating system.)

[Android Studio Install Instructions](https://developer.android.com/studio/install)

Install the Flutter plugin. You will go to File > Settings ... Plugins. Then, search the Marketplace
for flutter. Select the Flutter plugin and click Install.

For the Flutter plugin, you can follow the "Install the Flutter plugin" section on this page:
[Android Studio and IntelliJ setup](https://docs.flutter.dev/tools/android-studio)


## Flutter

Flutter will install everything you need to write both Dart and Flutter programs. You should be able to follow the
instructions on the website to both download and install it. It is all free, so you don't have to make any
purchases.

[Flutter Quick Install](https://docs.flutter.dev/install/quick)

If you want more detail or want to be more hands on, you can start at the main install page.

[Full Install Instructions Page](https://docs.flutter.dev/install)

# Exercise

Install Flutter and either Android Studio or Visual Studio Code.

Run "flutter --doctor" in your terminal / command window when you are done to make sure
that everything is working.
