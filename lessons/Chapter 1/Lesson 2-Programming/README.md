# Concepts to Learn

## Programming

"Computer programming or coding is the composition of sequences of instructions, called programs, that computers can follow to perform tasks." (From Wikipedia)

Read: [Wikipedia: Computer Programming](https://en.wikipedia.org/wiki/Computer_programming)

## Pseudocode

Pseudocode is like programming code, but not for the computer. The goal is to help you think
through logic the way a computer would understand it, but not have to be tied to all of the 
syntax rules of any particular computer language.

Read: [Science Direct: Pseudocode](https://www.sciencedirect.com/topics/engineering/pseudocode)

The cool part about Pseudocode is that you can do that now before you've even learned any
computer language or even how to code.

# Exercise

## Brushing Teeth

This was my first assignment in college as far as programming goes: write instructions for brushing teeth.

Can you write out how to brush your teeth in 20 steps? The goal is not to see how few steps you can make it but how much detail you can put
into the process. Remember, we aren't talking about AI now, so computers won't think and assume. They need every little detail and
do things step by step.

## Choosing a Car

Can you write out pseudocode for evaluating a car for purchase?

Write some logic that receives details about a car such as make, model, number of doors, number of seats, miles per galloon, etc. The
logic takes those details and returns yes or no as to whether it meets your needs and might buy it. Write your program in Pseudocode.
