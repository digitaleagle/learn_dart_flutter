# Concepts to Learn

## Dart

Dart is the programming language that this course will teach you.

I like Dart because it has modern features and allows you to accomplish more than older languages with less effort.
It is also important because it is the language that Flutter uses.

Glance through the Dart website because it lists some of the advantages and features of the language. It also gives
you some examples.

[Dart Website](https://dart.dev/)

The Overview may be difficult to understand, and don't worry if it doesn't all make sense. It's still worth skimming
through to see if you can pick up any details about Dart.

[Dart Overview](https://dart.dev/overview)

## Flutter

Flutter is a framework. It's a collection of tools that help build mobile and desktop applications. The big special
feature that Flutter offers is that you can create one application and Flutter will build from that an application
that can run on mobile phones (both Android and IOS), on desktop (both Windows, Mac, and Linux), and on web.

This course will focus first on Dart. You have to understand the principles of programming and the language first. Then,
we can add the concepts from Flutter to it.

Glance through the Flutter website. It gives an introduction to what Flutter is.

[Flutter Website](https://flutter.dev/)

Read through the FAQ to learn the answers to common questions about Flutter.

[Flutter FAQ](https://docs.flutter.dev/resources/faq)
